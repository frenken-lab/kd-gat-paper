# Data Schemas

CSV column contracts for paper tables. Generated by `KD-GAT/scripts/export_paper_data.py`.

## car_hacking.csv

Test performance on HCRL Car-Hacking dataset.

| Column | Type | Description |
|--------|------|-------------|
| method | str | Model name (e.g., "KD-GAT", "Proposed (GAT)", "Proposed (Fusion)") |
| accuracy | float | Test accuracy (0-1) |
| f1 | float | Test F1 score (0-1) |
| precision | float | Test precision (0-1) |
| recall | float | Test recall (0-1) |
| source | str | "ours" or "literature" (baseline rows from cited papers) |

## can_datasets.csv

Test performance on can-train-and-test S01–S04 subsets.

| Column | Type | Description |
|--------|------|-------------|
| method | str | Model name |
| dataset | str | "S01", "S02", "S03", "S04" |
| accuracy | float | Test accuracy (0-1) |
| f1 | float | Test F1 score (0-1) |
| source | str | "ours" or "literature" |

## ablation_kd.csv

Knowledge distillation ablation.

| Column | Type | Description |
|--------|------|-------------|
| model | str | "Teacher GAT", "Student GAT (w/ KD)", "Student GAT (w/o KD)" |
| dataset | str | Dataset name |
| accuracy | float | Test accuracy |
| f1 | float | Test F1 |
| params | int | Parameter count |

## ablation_gat_training.csv

GAT training strategy comparison.

| Column | Type | Description |
|--------|------|-------------|
| strategy | str | "Baseline", "Hard Sample Mining", "Curriculum", "Curriculum + Hard Samples" |
| dataset | str | Dataset name |
| accuracy | float | Test accuracy |
| f1 | float | Test F1 |

## ablation_fusion.csv

DQN fusion vs baseline fusion strategies.

| Column | Type | Description |
|--------|------|-------------|
| strategy | str | "GAT Only", "VGAE Only", "Average", "Weighted Average", "DQN Fusion" |
| dataset | str | Dataset name |
| accuracy | float | Test accuracy |
| f1 | float | Test F1 |

## model_params.csv

Architecture details for teacher and student models.

| Column | Type | Description |
|--------|------|-------------|
| model | str | "GAT Teacher", "GAT Student", "VGAE Teacher", etc. |
| param | str | Parameter name (e.g., "layers", "heads", "hidden_dim") |
| value | str | Parameter value |

## Source Mapping

Each CSV is derived from KD-GAT experiment outputs:

| CSV | KD-GAT Source |
|-----|--------------|
| car_hacking.csv | `experimentruns/hcrl_ch/eval_*/metrics.json` → `test.{gat,vgae,fusion}` |
| can_datasets.csv | `experimentruns/set_0{1-4}/eval_*/metrics.json` → `test.*` |
| ablation_kd.csv | Compare `gat_*_normal/` vs `gat_*_curriculum/` metrics.json |
| ablation_gat_training.csv | Compare training strategy variants in metrics.json |
| ablation_fusion.csv | Compare standalone vs fusion in `eval_*/metrics.json` |
| model_params.csv | `experimentruns/*/config.json` architecture fields |
