# Interactive Figure Data Contracts

Each Svelte figure imports a `data.json`. These are generated by `KD-GAT/scripts/export_paper_data.py`.

## umap/data.json

UMAP 2D projections of graph-level embeddings (10% sample of dataset).

```json
[{
  "x": -5.2,        // UMAP dimension 1
  "y": 3.1,         // UMAP dimension 2
  "label": 0,       // Binary label (0=normal, 1=attack)
  "attack_type": "Normal",  // Attack type string
  "confidence": 0.98,       // GAT classification confidence
  "graph_idx": 0             // Original graph index
}]
```

**Source:** `eval_*/embeddings.npz` → `gat_emb` + `gat_labels` + `gat_attack_types`.
Apply UMAP(n_components=2) to `gat_emb`. Sample 10% for file size.

**Expected size:** ~3K-5K records, ~200-400KB JSON.

## fusion/data.json

DQN fusion weight decisions per sample.

```json
[{
  "alpha": 0.2,              // Fusion weight (0=VGAE, 1=GAT)
  "label": 0,                // Ground truth
  "attack_type": "Normal",
  "vgae_score": 0.12,        // VGAE anomaly score
  "gat_confidence": 0.95     // GAT classification confidence
}]
```

**Source:** `eval_*/dqn_policy.json` → `alphas` + `labels` + `alpha_by_label`.
Cross-reference with `embeddings.npz` for attack types. Sample if >10K records.

**Expected size:** ~5K-10K records, ~300-600KB JSON.

## reconstruction/data.json

VGAE reconstruction error components.

```json
[{
  "composite_error": 0.45,   // Composite reconstruction error (MSE of continuous features)
  "node_error": 0.12,        // Node feature reconstruction MSE (vgae_error_recon)
  "canid_error": 0.33,       // CAN ID prediction cross-entropy (vgae_error_canid)
  "neighbor_error": 0.015,   // Neighborhood reconstruction BCE (vgae_error_nbr)
  "kl_error": 0.001,         // KL divergence (vgae_error_kl)
  "label": 0,
  "attack_type": "Normal"
}]
```

**Source:** `eval_*/embeddings.npz` → `vgae_errors` (composite) + `vgae_error_recon`,
`vgae_error_canid`, `vgae_error_nbr`, `vgae_error_kl` (per-component arrays). Sample 10%.

**Expected size:** ~3K-5K records, ~200KB JSON.

## cka/data.json

CKA layer similarity matrix (teacher vs student).

```json
{
  "matrix": [[0.95, 0.82, ...], ...],  // Shape: [teacher_layers, student_layers]
  "teacher_layers": ["Teacher L1", "Teacher L2", "Teacher L3", "Teacher L4", "Teacher L5"],
  "student_layers": ["Student L1", "Student L2"]
}
```

**Source:** `eval_*/cka_matrix.json` — direct copy, already in correct format.

**Expected size:** <1KB.

## attention/data.json

GAT attention weights for selected graphs (≤10 for visualization).

```json
[{
  "graph_idx": 42,
  "label": 1,
  "attack_type": "DoS",
  "nodes": [
    {"id": 0, "can_id": "0x1A0"}
  ],
  "edges": [
    {"source": 0, "target": 1, "layer_0_attention": 0.85, "layer_1_attention": 0.72}
  ]
}]
```

**Source:** `eval_*/attention_weights.npz` → per-sample edge_index + layer attention alphas.
Select ≤10 diverse graphs (mix of normal + attack types).

**Expected size:** ~50-100KB JSON.

## Build Pipeline

```
KD-GAT experimentruns/        →  scripts/export_paper_data.py  →  kd-gat-paper/
  eval_*/embeddings.npz                                             interactive/src/*/data.json
  eval_*/dqn_policy.json                                            data/*.csv
  eval_*/cka_matrix.json                                            figures/*.png (static fallback)
  eval_*/metrics.json
  eval_*/attention_weights.npz
  */config.json
```
